jhipster jhipster is a free and opensource application generator used to develop quickly a modern web application using angularjs and the spring framework overview jhipster provides tools to generate a project with a java stack on the server side using spring boot and a responsive web frontend on the client side with angularjs and bootstrap the term jhipster comes from java hipster as its initial goal was to use all the modern and hype tools available at the time today it has reached a more enterprise goal with a strong focus on developer productivity tooling and quality technology stack on the client side on the server side outofthebox autoconfigured tooling book a jhipster book is being written by matt raible the author of appfuse
