clanlib clanlib is a crossplatform c game sdk currently supporting microsoft windows macos and linux with partial support for mobile platforms it has full hardware accelerated graphics support through opengl and also a software renderer clanlib also helps in playing sound using the vorbis or mikmod libraries and has classes for collision detection guis xml networking and other things that may be helpful to a game programmer history the earliest known public release is in  version  the book introduction to c game programming published june  dedicates a chapter to learn how to use the clanlib library to make d games the book game programming with python lua and ruby published december  has a chapter about using clanlib together with ruby
