indoona indoona is a proprietary crossplatform instant messaging service indoona launched in  as wiphone is operated by tiscali an italian telecommunications provider and internetrelated services developer it is currently available as a mobile application on android ios and windows phone smartphones and as a website on web browsers by means of the webrtc technology indoona provides text messages graphics video and audio media exchanging free voip calls within the indoona service charged calls to landline telephones and mobile phones via a debitbased user account system indoona is also an open platform that makes people interact with web smart services and objects directly from the phone address book indoonas platform api are available for the developers the development stack currently supports rest and the sdk are available for java and php
