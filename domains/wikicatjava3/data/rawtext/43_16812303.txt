retrotranslator retrotranslator is a program written in java that converts java classes bytecode the source classes may use java  and java  features whereas the target will be compatible with java  and older environments retrotranslator supports all java  language features and part of the java  api on both jse  and jse  on older java environments only the java  features that dont depend on the new api are supported retrotranslator is based on the asm bytecode manipulation framework and the backport of the java  concurrency utilities see httpbackportjsrsourceforgenet  the stripes framework developers suggest to use retrotranslator in order to use stripes which extensively use java  features on a java  environment
