litec litec is a programming language for multimedia applications and personal computer games using a syntax subset of the c language with some elements of the c language its main difference to c is the native implementation of multimedia and computer game related objects like sounds images movies gui elements d and d models collision detection and rigid body physics litec executables are compiled instead of interpreted litec runs on bit and bit windows xp or vista operating systems litec claims to allow very fast programming with a minimum of code and easy access to nonprogrammers for this the developer provides a lesson workshop that especially deals with the game and multimedia related objects of the language litec supports the windows api and the component object model com therefore opengl and directx programs can directly be written in litec it has integrated the free a rendering engine examples the following litec program prints hello world then plays a movie file and exits the following litec program opens a d window and displays a spinning sphere features litec has the following differences to standard c litec supports rudimentary classes and function overloading but does not support advanced language concepts such as inheritance polymorphism or operator overloading history the litec language and compiler was originally developed in  by conitec inc for atari inc with the focus on creating computer games by nonprogrammers since  litec is also used for defining automatic trade algorithms in day trading software
