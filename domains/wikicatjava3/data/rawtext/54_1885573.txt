apache hivemind apache hivemind is an inversion of control ioc software project of the apache software foundation written in java it takes the form of a services and configuration microkernel in hivemind a service is an implementation of a java interface unlike other serviceoriented architectures soas hivemind is explicit about combining java code within a single jvm the hivemind project formerly a toplevel apache project was retired  april  ending its life in the apache attic repository its successor is considered to be tapestry ioc tapestry ioc a guicelike ioc container for java ee modelviewcontroller web applications
