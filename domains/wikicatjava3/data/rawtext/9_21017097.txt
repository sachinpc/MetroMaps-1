concurrent collections concurrent collections known as cnc is a programming model for software frameworks to expose parallelism in applications the concurrent collections conception originated from tagged stream processing development with hp tstreams tstreams around  hewlettpackard cambridge research lab developed tstreams a stream processing forerunner of the basic concepts of cnc concurrent collections for c concurrent collections for c is an open source c template library developed by intel for implementing parallel cnc applications in c with shared andor distributed memory habanero cnc rice university has developed various cnc language implementations based on their habanero project infrastructure
