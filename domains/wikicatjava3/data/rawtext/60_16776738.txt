jake jake is a java port of the gpl release of the quake ii game engine history the  version of jake was shown by the jogl team for javaone  to present an example of javaopengl interoperability jake has since been used by sun as an example of java web start capabilities for games distribution over the internet as of august  development is still active in the lwjake fork which removed jogljoal in favor of the lwjgl as of october  a fork using the current version of java opengl and java openal has been made available on a git repository the game can be played online launched as an applet or webstart the new enhancements allows jake to be run on mobile platforms as well due to utilizing jogls opengl fixed function emulation via opengl es or directly via opengl es joal enhancements further allows audio output on mobile devices the fork has been updated in june  for the latest jogl release alternative uses possibly because its code is based on java and thus is easy to modify jake has been used for several experiments architecture jake is a fulljava application it can use java opengl  java openal or lwjgl for its internal java lowlevel opengl library both single and multiplayer modes of quake ii are working and it is possible to use it both as an installed application or by java web start the performance of jake is on par with the original c version in some hardware configurations it is even better
