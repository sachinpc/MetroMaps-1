stormmq in computing stormmq is a message queuing service a form of messageoriented middleware that provides a message broker using the standard advanced message queuing protocol amqp stormmq is a hosted onpremises or cloud solution for machinetomachine message queuing using amqp stormmq limited provides an open secure platform and the necessary legal protections for data moved through the cloud the platform has clients developed by stormmq with third parties available as open source with code released under the mozilla public license
