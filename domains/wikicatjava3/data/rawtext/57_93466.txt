xalan xalan is a popular open source software library from the apache software foundation that implements the xslt  xml transformation language and the xpath  language the xalan xslt processor is available for both the java and c programming languages it combines technology from two main sources an xslt processor originally created by ibm under the name lotusxsl and an xslt compiler created by sun microsystems under the name xsltc a wrapper for the eiffel language is available
