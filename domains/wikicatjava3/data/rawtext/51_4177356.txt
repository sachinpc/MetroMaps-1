javanet javanet is a java technology related community website it also offers a webbased source code repository for java projects history javanet was announced by sun microsystems during javaone  in january  oracle announced that it will migrate javanet portal to project kenai codebase encouraging users to move their kenai projects to javanet in june  oracle announced that the javanet and kenaicom forges will be going dark on april   javapedia the javapedia project was launched in june  during the javaone developer conference it is part of javanet the project aims at creating an online encyclopedia covering all aspects of the java platform the javapedia project is openly inspired by wikipedia the prominent differences between wikipedia and javapedia include feature restrictions for example editing is open to registered users only software used twiki links camelcase is used and content licensing creative commons  attribution license
